Mockumentary: Dish Memory!

The Sudsy forgetfulness






The concept of this project is to produce a satire mockumentary. I aimed to create a ‘Horatian’ satire, about how long-term dishwashing causes memory loss according to a new study. This is an unrealistic study only for mockumentary purposes, aiming to emphasize on how false claims are easily believed even though the idea could be clearly absurd.
